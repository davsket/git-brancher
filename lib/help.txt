Brancher Help
=============

By default this transform the entry into a valid branch name, you can pass in any symbols incudin spaces

Valid branch names? http://stackoverflow.com/questions/3651860/which-characters-are-illegal-within-a-branch-name

Synopsis
    branch [any-text]

Commands
    help ............ shows this help

Examples
    branch
    | Will prompt you to input the text to create the branch
    branch MyProject-2992 [PROD] Duplicate phone number error on user forms
    | It will be transformed into MyProject-2992-[PROD]-Duplicate-phone-number-error-on-user-forms
    | If you select a prefix it will be created as <prefix>/MyProject-2992-[PROD]-Duplicate-phone-number-error-on-user-forms
